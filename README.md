# My NeetCode 150 Journey

Just a repo where I'm dumping my LeetCode solutions while prepping for placements. Following the [NeetCode 150](https://neetcode.io/roadmap) roadmap - big thanks to NeetCode for curating this list!

### What's Inside
- My raw solutions to problems
- Quick notes I'd jot down if someone asked me the question
- Test cases to verify things work
- Things I learned after solving each problem

### How I Use This
1. Solve a problem
2. Add it here with:
   - My first thoughts about the problem
   - The solution code (unchanged from how I first solved it)
   - Some test cases
   - What I learned from solving it

### Why This Exists

Placement season is stressful enough - this is just my way of:

- Keeping track of what I've solved  
- Noting down insights so I remember them later  
- Having all my solutions in one place  

Shoutout again to [NeetCode](https://neetcode.io) for the great problem selection!
